# ⏱ Stopwatch Web App

A simple stopwatch built using *HTML, CSS, and JavaScript*.  
It allows users to start, pause, reset, and track elapsed time.

---

## 🚀 Features
- Start / Pause functionality
- Reset timer
- Responsive design
- Lightweight and fast (no external libraries)

---

## 🛠 Tech Stack
- *HTML5*
- *CSS3*
- *JavaScript (ES6)*
